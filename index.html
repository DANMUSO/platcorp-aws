AWS Services Overview - ECS - ECR- CLUSTER - SERVICE

EC2

Amazon EC2 provides scalable computing capacity in the AWS cloud. It allows you to deploy and manage virtual servers with ease.

ECR

Amazon ECR is a fully managed container registry that makes it easy for developers to store, manage, and deploy Docker container images.

ECS

Amazon ECS is a highly scalable and high-performance container orchestration service that supports Docker containers.

ELB

Amazon ELB distributes incoming application traffic across multiple targets, such as EC2 instances, in multiple Availability Zones.
